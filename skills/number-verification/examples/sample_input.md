# AI Agent 赛道扫描 · 摘要稿

## 市场规模

全球 AI Agent 市场 2024 年规模约 51 亿美元，预计到 2030 年将达到 471 亿美元，CAGR 36.4%。
中国市场 2024 年约 12.5 亿美元，占全球 24.5%。

## 用户规模

ChatGPT 月活已突破 5 亿，其中付费用户约 1500 万。
全球互联网用户 53 亿，渗透率约 65%。

## 商业模式

主流 Agent SaaS 定价 $20-50/月/座，企业版 $200-500/座。
按 100 万付费用户、平均 $25/月计算，单家头部公司年收入可达 3 亿美元。

## 技术指标

最新 GPT-5 模型在 MMLU 评测中得分 91.2%，比 GPT-4 的 86.4% 提升 5.6 个百分点。
长上下文模型已支持 200 万 token 输入，相当于一本中等长度小说。
